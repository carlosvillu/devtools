Inline SVG icon rendered from the curated Lucide glyph set — the devtools icon system. Use anywhere a small stroke icon is needed; inherits `currentColor`.

```jsx
<Icon name="copy" size={16} />
<Icon name="git-branch" size={18} strokeWidth={1.75} style={{ color: "var(--accent)" }} />
```

Names: copy, check, chevron-down/right/up, x, trash, clock, arrow-down, corner-down-right, alert-triangle, info, shield, loader, terminal, key, braces, link, hash, calendar, type, eye, eye-off, reopen, search, git-branch. Default size 16, strokeWidth 2. All icons are 24×24 viewBox and scale with `size`.
