Centered zero-state with an icon, title, description and optional action. Fills empty regions (empty history, no matches) so the product never shows a blank panel.

```jsx
<EmptyState icon="clock" title="Sin historial todavía"
  description="Las entradas que analices con la cuenta iniciada aparecerán aquí."
  action={<Button variant="secondary">Analizar algo</Button>} />
```

Props: `icon`, `title`, `description`, `action`.
