TextDecoder (por defecto)      → primer code point U+7b (len 7) ← SE COME EL BOM
TextDecoder {ignoreBOM:true}   → primer code point U+feff (len 8)
JSON.parse: Buffer=no parsea · default=PARSEA · ignoreBOM=no parsea
